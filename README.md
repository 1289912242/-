# 🌤️ 极简 · 气象台

> 一个简洁、优雅、开箱即用的天气查询工具，支持全球城市实时天气与未来 5 天预报。

---

## ✨ 项目简介

**极简气象台** 是一款轻量级天气查询应用，采用纯前端技术构建，无需后端服务器即可运行。它整合了**心知天气**与 **wttr.in** 双数据源，自动降级保障，让您随时获取准确、美观的天气信息。

## 🎯 核心功能

- 🔍 **城市搜索** — 输入任意城市名称，一键查询实时天气
- 📍 **自动定位** — 基于浏览器 Geolocation API，快速定位当前位置
- 📊 **实时天气** — 温度、湿度、风速、体感温度、高低温范围
- 📅 **5 天预报** — 展示未来 5 天天气趋势，含图标与温度区间
- 🕒 **搜索记录** — 自动保存最近 10 条搜索记录，点击快速复用
- 🌐 **双源保障** — 优先使用心知天气，失败时自动降级至 wttr.in
- 📱 **响应式设计** — 完美适配手机、平板、桌面端

## 🛠️ 技术栈

| 技术 | 说明 |
|------|------|
| HTML5 + CSS3 | 语义化结构 + 毛玻璃（Glassmorphism）UI |
| JavaScript (ES6+) | 原生 JS，无第三方依赖 |
| Font Awesome 6 | 天气图标与 UI 图标库 |
| 心知天气 API | 主数据源（需申请密钥） |
| wttr.in API | 备用数据源（无需密钥，免费 CORS 支持） |
| LocalStorage | 本地存储搜索历史与上次查询城市 |

## 🔧 快速开始

### 1. 获取心知天气密钥（可选）

访问 [心知天气官网](https://www.seniverse.com/) 注册账号，在控制台获取免费 API 密钥。

### 2. 配置密钥

打开 `index.html`，找到以下代码并替换：

```javascript
// 心知天气配置（请替换为您的有效密钥）
const SENIVERSE_KEY = 'YOUR_VALID_SENIVERSE_KEY';  // ← 替换为你的有效密钥
```

> ⚠️ 若不配置密钥，程序将自动使用 **wttr.in** 作为数据源，功能完全可用。

### 3. 运行项目

- **方式一（推荐）**：使用 VS Code 的 Live Server 插件打开 `index.html`
- **方式二**：直接双击 `index.html` 在浏览器中运行
- **方式三**：部署到任意静态托管服务（Vercel / Netlify / GitHub Pages）

## 🌐 API 说明

### 主数据源 — 心知天气（需密钥）

| 接口 | 用途 |
|------|------|
| `v3/weather/now.json` | 实时天气 |
| `v3/weather/daily.json` | 5 天预报 |

### 备用数据源 — wttr.in（无需密钥）

| 接口 | 用途 |
|------|------|
| `/{city}?format=j1&lang=zh` | 实时天气 + 预报 |

## 🎨 设计亮点

- **毛玻璃质感** — 背景模糊与半透明叠加，营造沉浸式视觉体验
- **动态图标映射** — 根据天气文本自动匹配 Font Awesome 图标
- **微交互动效** — 按钮悬停、卡片抬起、Toast 提示，提升操作反馈
- **暗光兼容** — 柔和渐变背景，长时间使用不刺眼
